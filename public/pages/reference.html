Endpoints ‐ API v1

  • GETapi v1 lookup
  • Parameters:

  • q = The comma separated RGB Color (0-255). Values will be clamped if not in range
  • match (optional) = Number of matches to be found. Default value is taken as 1 if not provided

  • Example call:

  • https://color-tools.herokuapp.com/pages/reference.html

  • JSON result format:

  • MatchInfo
    • status : "success" | "failure"
    • cause : "error message if failure"
    • given_color : "rgb representation of the processed input color"
      • red : "the red value 0-255"
      • green : "the green value 0-255"
      • blue : "the blue value 0-255"
    • matches : "an array of matched colors"
      • Array elements format
      • color : "rgb representation of the matched color"
        • red : "the red value 0-255"
        • green : "the green value 0-255"
        • blue : "the blue value 0-255"
      • hex : "hex-string representation of the matched color eg: 0xffff"
      • name : "the matched color name"
